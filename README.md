## Как запустить проект

1. Откройте нужный проект (`ConsoleApp12`, `ConsoleApp13`, и т.д.) в Visual Studio или через CLI.
2. Убедитесь, что установлен `.NET 8 SDK`.
3. Запустите выбранный проект:

```bash
dotnet run --project ConsoleApp12/ConsoleApp12
```

Замените путь на нужный проект из таблицы выше.
